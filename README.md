# 💱 Real-Time Currency Converter

A Python application that converts currencies in real-time using ExchangeRate-API. Perfect for learning API integration and building fintech applications.

## ✨ Features

- ✅ Real-time currency conversion using live API
- ✅ Support for 160+ currencies worldwide
- ✅ Multi-currency conversion (one amount to many currencies)
- ✅ Conversion history saved to JSON file
- ✅ Clean CLI interface with emojis
- ✅ API error handling (no internet, invalid currencies)
- ✅ Rate caching to avoid unnecessary API calls
- ✅ Input validation and case-insensitive currency codes

## 📋 Prerequisites

- Python 3.7 or higher
- Internet connection for live rates
- (Optional) Free API key from ExchangeRate-API

## 🚀 Installation

1. **Clone the repository**

git clone https://github.com/yourusername/currency-converter.git
cd currency-converter

2. Install dependencies


pip install -r requirements.txt

3.Run the application

python currency_converter.py
